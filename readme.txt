== Description ==

Member Biography is simple Member name & with details plugin. User can very easily upload unlimited member.

Features:

* There is no limitation to the number of member name.


== Installation ==

Steps

1. Upload 'biography' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can find the "biography" menu in Admin Panel.
4. Place shortcode '[MemberAll]' in your post(s) or page.
